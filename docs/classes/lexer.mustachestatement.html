Class MustacheStatement

The mustache statement parses the content inside the curly braces. Since the statement can be in multiple lines, this class seeks for more content unless closing braces are detected.

const statement = new MustacheStatement(1)
statement.feed('Hello {{ username }}!')

console.log(statement.props)
{
  name: 'mustache',
  jsArg: ' username ',
  raw: 'Hello {{ username }}!',
  textLeft: 'Hello ',
  textRight: '!'
}
